## 项目简介

这是一个运行在 **micro:bit V2** 上的双人合奏程序：

- **单人模式**：默认启动，一块 micro:bit 独立播放完整曲目
- **双人合奏**：两块 micro:bit 通过 **无线电** 连接，分别演奏不同声部（A / B）
- **按键控制**：A 键快退 / B 键快进 / LOGO 进入角色选择 / 晃动触发彩蛋

## 使用方法

### 第一步：获取代码
- 将本仓库中的 `main.py` 文件内容**全选复制**

### 第二步：烧录到 micro:bit
1. 打开 [MakeCode 网页编辑器](https://makecode.microbit.org)
2. 点击 **“新建项目”**，选择 **“Python”** 编程方式
3. 删除编辑器中的所有默认代码，**粘贴**你刚才复制的代码
4. 将 micro:bit 连接到电脑，点击 **“下载”** 即可

### 第三步：开始演奏（双人合奏流程）
| 步骤 | 操作 |
| :--- | :--- |
| ① | 两块 micro:bit 分别烧录好程序，开机 |
| ② | 默认进入**单人模式**，单机会自动播放 |
| ③ | 单击 **LOGO 触摸键**，屏幕提示选择角色 |
| ④ | 按 **A 键** 选“A声部”，按 **B 键** 选“B声部”（两块板子必须选不同角色） |
| ⑤ | 两块板子会自动通过无线电连接，连接成功后屏幕会显示对方名称 |
| ⑥ | 在 **A声部** 的板子上按一下 **A 键**，双人合奏开始！ |

### 按键功能一览
| 按键 | 单人模式 | 双人模式 | 角色选择 | 连接状态 |
| :--- | :--- | :--- | :--- |:--- |
| **A** | 快退 5 个音符 | 无操作 | 选择角色 | 触发合奏 |
| **B** | 快进 5 个音符 | 触发“断开连接”询问 | 选择角色 | 询问断开连接（再按A确认） |
| **LOGO** | 进入角色选择 | 无操作 | 返回单人模式 | 无操作 |
| **晃动** | 触发随机彩蛋 | 触发随机彩蛋 | 无操作 | 无操作 |

## 屏幕状态说明

| 屏幕显示内容 | 含义 |
| :--- | :--- |
| 进度条（点阵从左到右点亮） | 当前播放进度（仅演奏时显示） |
| `PRESS BUTTON TO CONFIRM YOUR ROLE` | 请按 A 或 B 选择声部 |
| `NAMExxx WAITING FOR CONNECTION` | 等待另一块 micro:bit 连接 |
| `CONNECTED, PRESS A TO PLAY...` | 已连接，按 A 开始合奏 |
| `ARE YOU SURE TO CUT DOWN CONNECTION?` | 按 A 确认断开，按 B 取消 |
